How to get Premium
Premium is granted by the bot owner directly.
  1. Join the Onyx support server on Discord.
  2. Open a ticket in #premium-requests.
  3. The bot owner will run >premium add <guildId>.
  4. Premium activates instantly — reload this page to verify.
